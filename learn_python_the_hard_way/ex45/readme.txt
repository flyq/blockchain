這是一個基於python的， 便於擴展的， 通過文字與玩家進行交互的初級遊戲。

标准：
1、非无限循环，一律用for;
2、每一个if语句，必须包含一个else;
3、如果else永远不应该执行，必须在else后使用die函数;
4、if语句最多嵌套2层;
5、布尔测试不超过两个关键字;
6、每个段落都需要分行，每个操作符后都需要空格；
7、代码里每行不超过80个英文字符或者40个汉字，包括格式；
8、提示输出每行中午最大30个，英文最大60个；分割线用14个-
9、在class里面的method，用动词准确命名。
10、函数尽量小巧
11、类用驼峰命令、函数用_命名
